Barnyard Animals Plan

Joshua Dean

WDD 330

Overview

Purpose

A interactive barnyard where you can view and see all different types of farm animals, a picutre of them, and some info and facts on them.

This gives people who want quick and easy access to farm animal information and facts easily accessible to them.

Initial Module List

  • Main

    (The main script that will run and call all the other modules as need be)

  • Util

    (A module for useful util functions)

  • AnimalList

    (A module for listing the farm animals on the page so they can be chosen)

  • AnimalDetails

    (A module for showing the details of a selected animal)

  • FavoriteAnimal

    (A module enabling favoriting an animal so you can quicly view animals you have or want on your farm)

Schedule

  • Project Week 1: Complete The Header, Footer, and Navigation for all the tabs, as well as the design for the website
  • Project Week 2: Complete AnimalList and AnimalDetail modules and allow for users to look at the list of farm animals and see their details
  • Project Week 3: Complete the rest of the modules and add the Favorite feature, as well as cleaning up css and debugging any last errors

Audience

Farm hands and aspiring animal scientists.
